Hardware Random Number Generator words

Hardware random number generators are initialized at boot time, and can be used as is afterwards. All currently supported platforms support hardware random number generation.

Note that hardware random number generation can be very slow, so it is highly recommended that one use it to seed a pseudorandom number generator such as TinyMT32 rather than directly using it to generate random numbers in sequence.
